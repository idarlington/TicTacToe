# TicTacToe

## Introduction
[TicTacToe](https://en.wikipedia.org/wiki/Tic-tac-toe) console game implemented in `Scala`

## How to run
- Clone project
- [Install sbt](https://www.scala-sbt.org/1.x/docs/Setup.html)
- Run `sbt run`

## How to play
Enter input into the console as suggested

## Running tests
```shell
sbt test
```
